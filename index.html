Windows Installation Guide

  1. Download the game

    Click the button below to download the game for Windows.

    Download for Windows
  2. Extract the ZIP file

    Right-click on the downloaded file and select "Extract All".

  3. Run the Installer

    Double-click on "setup.exe" and follow the installation steps.

  4. Start the Game

    After installation, launch the game from your desktop shortcut.

Linux Installation Guide

  1. Download the game

    Click the button below to download the game for Linux.

    Download for Linux
  2. Extract the files

    Open a terminal and run:

    tar -xvzf mygame-linux.tar.gz
  3. Run the game

    Navigate to the game folder and start the game with:

    ./start_game.sh
