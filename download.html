Windows Installation Guide

  1. Download the ZIP file and extract it.
  2. Run `setup.exe` and follow the installation steps.
  3. After installation, start the game from the desktop shortcut.
  4. Enjoy your adventure in Iroas!

Linux Installation Guide

  1. Download the `.tar.gz` file and extract it using `tar -xvzf mygame-linux.tar.gz`.
  2. Navigate to the game folder in the terminal.
  3. Run the game with `./start_game.sh`.
  4. Enjoy your adventure in Iroas!
